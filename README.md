Coco
----

## Features

* OTP Certification

* LDAP User Manage

* Command Record


## Install

依赖:
python版本为3.6.1, 并软连为python3, 如: ln -s 原路径 /usr/bin/python3
virtualenv为python3.6.1所对应的版本, 并软连为virtualenv, 如: ln -s 原路径 /user/bin/virtualenv。

```
cd install
./control install
```


## Usage

```
cd install

# 启动
./control start

# 关闭
./control stop
```
